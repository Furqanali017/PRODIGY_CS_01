## ‚öôÔ∏è How the Caesar Cipher Works
- Each alphabetical character in the message is shifted forward by a fixed number (shift value).
- Uppercase and lowercase letters are handled separately.
- Non-alphabetical characters (spaces, numbers, symbols) remain unchanged.
- Decryption is performed by reversing the shift value.
